# ElimintionGame
This is a terminal-based strategy game inspired by classic mathematical Nim-style puzzles, 
where players compete to avoid being the one to remove the final tile from the board. Built in Python, 
the game features a dynamic grid system supporting custom board sizes up to 26x30, 
a "Single Player" mode against an AI that utilizes randomized move-length logic, and a "Multi Player" mode for local competition. 
With robust input validation, colored terminal output, and a built-in hint system that suggests optimal moves.
Players can further customize their experience by toggling random board generation and 
selecting from multiple difficulty levels that influence tile density and AI behavior.
